# @tecsinapse/react-web-kit

Since this package is based on `react-native` components, we need to use `react-native-web` as compatibility layer.

### Requirements

We recommend to bootstrap the project using [vitejs](https://vitejs.dev/) with `react-swc-ts` template. After following initial steps, add the following libraries:

```shell
pnpm add @tecsinapse/react-web-kit react-native-web
```

### Configure vite

In your `vite.config.ts` add the alias:

```javascript
export default defineConfig({
  resolve: {
    alias: {
      'react-native': 'react-native-web',
    },
  },
});
```

### Icons

You will need to copy the font files from **react-native-vector-icons** to your public folder. The base icons used on the package are `ionicons` and `material-community`,
all desired icons need to be copied.

Create a folder named `fonts` inside `public` folder and then run this shell copy (bash example):

```shell
cp node_modules/react-native-vector-icons/Fonts/* public/fonts
```

### Fonts

Import **Lato** family from Google fonts.

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Lato:wght@400;700;900&display=swap"
  rel="stylesheet"
/>
```

### Providers

Add to your `index` file:

```jsx
import { ThemeProvider, lightTheme } from '@tecsinapse/react-web-kit';

export default function Index() {
  return (
    <ThemeProvider theme={lightTheme}>
      <App />
    </ThemeProvider>
  );
}
```
