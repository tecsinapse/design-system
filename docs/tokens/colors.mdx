# Colors

You can use colors tokens with any applicable tailwind prefix, like background, border, text and others.

### NOTE: `text` colors works only with `text` prefix. You can apply all colors to text, but not "text only" to other prefixes.

Example: `bg-success-light`

Example (Text): `text-medium`
